# Endereçamento IP e a Camada de Rede

## Introdução
A camada de rede é responsável por permitir a comunicação entre dispositivos localizados em diferentes redes. Um dos principais protocolos utilizados nessa camada é o IP (Internet Protocol), que identifica de forma lógica cada dispositivo conectado à rede.

Este trabalho tem como objetivo apresentar os conceitos básicos de endereçamento IP e sua aplicação prática em um cenário de rede fictício.

---

## Camada de Rede
A camada de rede é a terceira camada do modelo OSI e tem como função principal o roteamento de pacotes de dados entre redes distintas. Ela garante que a informação enviada por um dispositivo chegue corretamente ao seu destino.

---

## Endereço IP
O endereço IP é um identificador numérico atribuído a cada dispositivo conectado a uma rede. Ele pode ser classificado como público ou privado.

### IP Público
Utilizado para comunicação direta com a internet, sendo fornecido por provedores de serviço.

### IP Privado
Utilizado em redes internas, não sendo acessível diretamente pela internet.

---

## Configuração de Rede
Uma configuração básica de rede IP é composta por:
- Endereço IP
- Máscara de Sub-rede
- Gateway Padrão
- Servidores DNS

Esses elementos permitem a comunicação interna e o acesso à internet.

---

## Conclusão
O estudo do endereçamento IP é fundamental para a compreensão do funcionamento das redes de computadores. A aplicação prática desses conceitos permite maior entendimento sobre a organização e comunicação entre dispositivos em uma rede local.
